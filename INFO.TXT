INFORMATION THAT ONE MIGHT FORGET

Pin values are integers. Integer pin values corresponding to GPIO pins are the same value (e.g. 18 = GPIO 18).

// HUB75E assumed pinout (source: MatrixPanel-DMA Library)
// R1 | G1
// B1 | GND
// R2 | G2
// B2 | E
//  A | B
//  C | D
// CLK| LAT
// OE | GND
